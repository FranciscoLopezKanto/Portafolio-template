# Mi Portafolio

Este es mi portafolio personal desarrollado con **React**, **Vite**, **TailwindCSS** y **TypeScript**.

## Requisitos previos

Asegúrate de tener instalado en tu equipo:

- [Node.js](https://nodejs.org/) (versión 16 o superior recomendada)
- [Yarn](https://yarnpkg.com) o [npm](https://www.npmjs.com/)

## Clonar el repositorio

```bash
git clone https://github.com/TU-USUARIO/TU-REPO.git
cd TU-REPO
```
*(Reemplaza `TU-USUARIO` y `TU-REPO` por tu usuario y nombre de repositorio)*

## Instalación de dependencias

Con **Yarn**:
```bash
yarn install
```

Con **npm**:
```bash
npm install
```

## Ejecución en modo desarrollo

Con **Yarn**:
```bash
yarn dev
```

Con **npm**:
```bash
npm run dev
```

Luego abre en tu navegador:
```
http://localhost:5173
```

## Construir para producción

Con **Yarn**:
```bash
yarn build
```

Con **npm**:
```bash
npm run build
```

## Previsualizar la versión de producción

Con **Yarn**:
```bash
yarn preview
```

Con **npm**:
```bash
npm run preview
```
